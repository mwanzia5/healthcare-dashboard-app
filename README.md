# HealthSync Dashboard

> A secure, modern healthcare dashboard for managing patient records, scheduling appointments, and viewing detailed medical histories with a focus on data privacy and a clean medical-blue aesthetic.

## Features

- **Patient List with Vitals Summaries** — Interactive table displaying patient records with quick-view vitals badges (BP, heart rate, temperature, SpO₂)
- **Appointment Scheduling Calendar** — Interactive calendar for viewing, creating, and managing patient appointments with drag-and-drop functionality
- **Medical History Modal** — Detailed modal view for patient medical history, allergies, medications, and visit notes
- **Data Privacy Visuals** — Prominent privacy indicators (lock icons, encryption status, compliance badges) throughout the interface
- **Dashboard Stats Overview** — Real-time cards showing total patients, upcoming appointments, critical alerts, and average wait time
- **Responsive Medical Design** — Clean "medical-blue" color palette with professional spacing and typography optimized for clinical workflows
- **Secure Sidebar Navigation** — Persistent sidebar with quick access to Patients, Calendar, Records, and Settings
- **Advanced Search & Filtering** — Filter patients by status, search by name or ID, and sort by last visit date

## Pages

| Route | Description |
|-------|-------------|
| / | Main Dashboard — Overview with stats, patient list, and calendar |
| /patients | Patient Management — Enhanced list with filtering and detailed actions |
| /calendar | Full Calendar View — Complete appointment scheduling interface |

## Tech Stack

- [Next.js 15](https://nextjs.org) — React framework with App Router for fast navigation
- [Tailwind CSS](https://tailwindcss.com) — Utility-first CSS framework for rapid UI development
- [Shadcn UI](https://ui.shadcn.com) — Accessible component library (tables, modals, cards, badges)
- [TypeScript](https://www.typescriptlang.org) — Type safety for patient data and medical records
- [Lucide React](https://lucide.dev) — Medical and healthcare icons (heart, thermometer, shield, calendar)
- [React Big Calendar](https://github.com/jquense/react-big-calendar) — Interactive calendar component for appointments
- [date-fns](https://date-fns.org) — Modern date utility library for formatting and manipulation

## Getting Started

npm install
cp .env.example .env.local
# Fill in your environment variables in .env.local
npm run dev

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| NEXT_PUBLIC_API_BASE_URL | Optional | Backend API base URL for patient data (mock data used if not set) |
| NEXT_PUBLIC_ENCRYPTION_ENABLED | Optional | Set to "true" to show encryption status visuals (default: true) |
